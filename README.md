<h1 align="center">AutoRetadType</h1>

> 레포 이름에 Retard 적었다가는 2번째로 정지 당할거같아서 Retad로 표기함
* 파이썬 자동 장애타 프로그램

## 사용법
* **main.py**
```py
CHANNELID = 1371020652227592255 # 메세지를 보낼 채널 아이디
TOKEN = "" # 메세지를 보낼 계정의 토큰
DELAY = 3 # 메세지 딜레이 (초)
```
